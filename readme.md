#TITLE

Timed Quiz Game with Instant Remarks

#DESCRIPTION

This project is an interactive Quiz Game where players answer questions and after each question, the game provides an instant remark (Correct/Incorrect).At the end , the final 
score and will be displayed.

#FEATURES

--> Instant remark after each question

--> Score Calculation

--> Randomized questions

--> Multiple Category(Optional)
